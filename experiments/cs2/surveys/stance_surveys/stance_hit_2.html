One-sidedness in Texts

About this HIT:
  1. Objective
    • Given two texts A and B, determine which text is more one-sided than the other.

  2. Instructions
    • An argumentative text wants to convince you of something. Some texts are more one-sided than others.
    • We will present you two texts on a controversial topic side by side (A and B), each including four arguments.
    • An argument can be either "pro" or "con" regarding the controversial topic. For example:


    • We call a text one-sided if it includes more pro arguments than con arguments (or the other way around).
    • We call a text more one-sided than another if it has more pro arguments than the other text (or con arguments).
    • In Example 2, text A is more one-sided than text B because text A has more pro arguments than con arguments, while text B has equal number of pro arguments and con arguments.
    • In Example 3, neither text is more one-sided because both have the same number of con arguments.
