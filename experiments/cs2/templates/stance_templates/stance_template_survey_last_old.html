One-sidedness in Argumentation Survey

About this HIT:
  1. Context

    2. An argument wants to convince you of something. But not all arguments are the same. Some are more one-sided than others in terms of how they try to convince you.

  2. Objective

    3. Given two different texts (A and B) determine which one is more one-sided. Completely read each text and think about which one is more one-sided than the other (Note: it is possible that both are equally one-sided).

  3. Disclaimer

    4. You will not be compensated if:

    1. You fail reading all the content, the system will detect if you don't read all the content
    2. You don't answer all the questions.
    3. You participated more than one time. You can answer this survey only one time. The focus of the experiment is to get different participants. Therefore, we need unique participants for this study. The system will reject your work if it finds that you already participated . So please to avoid any impact on your approval rating, answer this survey only once .

  4. How to answer this survey?

    1. The survey is comformed of six parts. We will present you examples comformed by two texts that discuss a controversial topic side by side (A and B, horizontally aligned) with four sentences each one. We call these sentences arguments:

      Example 1

      On the one hand, if an argument agrees with a controversial topic, we say that the argument has a pro stance regarding the topic. On the other hand, if the argument disagrees with the topic, we conclude that the stance of the argument is con. For example:

      Example 2

      In the above examples, you can see highlighted in green the arguments that support (or agree with) "compulsory vaccination". Arguments that disagree with "compulsory vaccination" are highlighted in red.

      We call a text one-sided, if it contains more pro arguments than con arguments (and vice-versa). For instance, example 1, is one sided because text A has more pro stances than text B. Example 2 is not one-sided, because all the stances of its arguments disagree with the topic.

      In the real cases you will not see the highlighted examples.

    2. Read completely both argumentations and then proceed to answer all the questions. You will not be allowed to continue if you do not answer all the questions. Your participation will not be approved if you dont read all texts, paragraphs and questions. Your work will be rejected if you answer or participate in this survey more than one time.
    3. Make sure you read all the texts, and associated paragraphs , so you can answer the the questions.
